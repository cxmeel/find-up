interface FindUpAllOptions {
ignoreCase: boolean?--

Whether to ignore case when comparing names

className: string?--

The class name of the instance to find

isA: string?--

Performs an :IsA check on each child

predicate: function?--

A function that should return true if the instance is a match

}

Extends GetAncestorsOptions.

Default options:

{
  ignoreCase = nil,
  className = nil,
  isA = nil,
  predicate = nil,
}

The instance to start searching from

The name of the instance to find

The options to use when searching

The instance found, or nil if not found

Returns the closest instance in the ancestry of self that satisfies the given options.

  className = "Folder",
})

FindFirstSibling​

</>
FindUp.FindFirstSibling(
self: Instance,--

The instance to start searching from (i.e. the child whose sibling you want to find)

name: string,--

The name of the sibling to find

options: FindUpAllOptions?--

The options to use when searching

) → Instance?--

The sibling found

Searches for the first sibling of self with the given name. This is a shortcut for using FindUp with stopAt set to self.Parent, and includeStopAt set to true. 

local sibling = FindFirstSibling(script, "ShopGuiFrame")

  stopAt = game.Workspace,
  includeSelf = true,
  includeStopAt = true,
})
